# vite-
- vp env : 実行環境まわりの確認/整備
- vp install : 依存インストール
- vp dev : 開発サーバー起動
- vp check : format + lint + type check を一括実行
- vp build : 本番ビルド
- vp test : テスト実行
- vp run : スクリプト実行（モノレポも意識された実行）
- vp pack : ライブラリ/配布向けパッケージ作成
